HTML

  • The head element contains mostly invisible infmormation about the webpage.
  • The body element contains the elements that are visible to the user
  • The body contains 3 elements which are header, main and footer.
  • The header usually contains the navigational bar and logo.
  • The Body contains the main elements of the page.
  • The footer contains information displyed at the bottom of the page such as the copyright.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Color can be added by typing the name like "red, blue, ..." or, more commonly in HEX code.
  • CSS can be added directly on the html file or on a seperate file usualy named style.css but needs to be linked to the html to work.

Git

  • git status: checks what branch we are currently on
  • git checkout main: Brings you back to main branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all filess in the working branch.
  • git commit -m "short descriptive message": associates a message with our commit.
  • git pull origin main: checks of current branch is up to date.
  • git push origin branch-name: pushes changes onto branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a list of related content like, sting,numbers, booleans,...
  • A function helps us not to repeat code
  • We use "" to indicate it is a string (text).
  • Chrome DevTools console is used to check our code.
